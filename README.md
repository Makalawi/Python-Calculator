# 🧮 Python Calculator

A simple command-line calculator built with Python.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ❌ Prevents division by zero

## Requirements

- Python 3.x

## How to Run

```bash
python calculator.py
```

## Example

```
Simple Calculator

First number: 20
Operator (+, -, *, /): *
Second number: 5

Result: 100
```
